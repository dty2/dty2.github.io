## 快速入手
### 安装
boost库是c++第三方库，开源，跨平台，被称为c++“准”标准库
### 编译
```
tar -xzvf 
cd boost_1_84_0
./bootstrap.sh # 准备编译环境
./b2 install # 使用b2安装编译和boost库
```
